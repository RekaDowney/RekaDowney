执行 Shell 脚本

　　Shell脚本可以通过两种方式执行：

  • /bin/bash /path/to/shell/script通过/bin/bash解释器直接执行脚本。
  • chmod +x /path/to/shell/script授予可执行权限之后直接使用/path/to/shell/script命令即可执行该脚本

　　注意：Linux对Shell脚本的后缀名并没有特殊性要求，使用.sh作为后缀只是为了方便识别Shell脚本。
